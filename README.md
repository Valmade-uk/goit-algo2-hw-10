Design and Analysis of Algorithms - Greedy algorithm for set cover

Домашнє завдання до модуля **«Алгоритми штучного інтелекту»**.

## Опис
Реалізація жадібного алгоритму для задачі **покриття множини (Set Cover)**: призначення викладачів на предмети так, щоб:
- покрити **всі** предмети з множини;
- мінімізувати кількість обраних викладачів;
- на кожному кроці обирати викладача, який покриває **найбільше** ще непокритих предметів;
- якщо кандидатів кілька — обирати **наймолодшого** (за потреби можна додати додатковий тай-брейк).

## Вхідні дані
- Множина предметів: `{'Математика', 'Фізика', 'Хімія', 'Інформатика', 'Біологія'}`
- Список викладачів (ім’я, прізвище, вік, email, множина предметів які може викладати)

## Вихідні дані
Список обраних викладачів із заповненим `assigned_subjects` (які предмети їм призначено).
Якщо покриття неможливе — виводиться повідомлення про це.

## Примітки
- Алгоритм є жадібним (greedy), тому він знаходить хороше рішення швидко, але в загальному випадку це не гарантує глобально оптимальний розв’язок для довільних наборів.
- Для даних із завдання алгоритм покриває всі предмети та обирає мінімально необхідну кількість викладачів згідно з правилами вибору.
